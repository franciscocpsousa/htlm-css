2 - Criar código base 
3 - Title "Página02"
4 - tag <h1> Esta é a minha segunda página
5 - Tag <p> A segunda página está aqui
6 - Volta pra página 1
7 -  Criar tag <p> Esta é a primeira página do site. Se você quiser pode acessar também a minha <a href= "ctrl+space">Segunda página</a></p>
obs : quando eu indico um link interno eu não preciso indicar a URL inteira ,contanto que a pasta esteja na mesma página do index.html.
8 - Criar um link na página 02 que vai pra primeira página.
9 - Na página 02 criar tag <p> Esta é a minha primeira página 
10 - Na página 02 cria tag <p> <a href="ctrl+space"> Voltar para a primeira página
11 - Acrescentar também rel="next"(na pag01) ou "prev"(na pag002) e assim sucessivamente
obs : rel = next/prev ajuda nos mecanismos de busca. Aparentemente não acontece nada mas 
existe uma semântica.
12 - Inserir tag <p> abaixo da frase "A segunda página está aqui"
13 - <p> Esta página é um oferecimento de Hostinet 
14 - acrescentar target =blank  ref= nofollow
obs : "nofollow" significa que você não dá o seu aval. Eu não quero que o mecanismo de 
busca perpetue o meu essa busca.
CRIANDO UMA TERCEIRA PÁGINA
1 - Criar nova pasta dentro da pasta de arquivos com nome "noticias"
2- No VSC clica na pag003 > new file > base code > title=Terceira página > h1 = Esta é a terceira página
3 - na página index <p>Você também pode acessar nossa <a href="noticias/pag003.html" target="_blank" rel="next">página de notícias</a></p>
4 - Na página index foi alteraro target= next para _self
<p>Esta é a primeira página do site. Se você quiser , pode acessar também a minha <a href="pag002.html" referrerpolicy="next" target="_self">Segunda página</a></p>
<p>Você também pode acessar nossa <a href="noticias/pag003.html" target="_blank" rel="next" target="_self">página de notícias</a></p>
obs : recomenda-se abrir páginas do mesmo site na mesma aba.
5 - Criar uma tag <p> na pag003
<p><a href="../index.html">Voltar para a primeira página</a></p>
obs :   ../  = duas pastas anteriores
comentário : se precisar voltar 2,3,4 pastas em um link é só continuar usando a sequência de"../../../arquivo.html" para cada sequencia de ../ o código volta uma página.
